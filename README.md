# Plant Leaf Disease Detection

A deep learning and Django-based web application for detecting plant leaf diseases from uploaded images.

## Features
- Upload plant leaf images
- Detect diseases using a trained deep learning model
- User-friendly Django web interface
- Displays disease prediction results

## Technologies Used
- Python
- Django
- TensorFlow/Keras
- OpenCV
- HTML/CSS
- SQLite

## How to Run
1. Clone the repository
2. Install requirements:
   pip install -r requirements.txt
3. Run:
   python manage.py runserver
