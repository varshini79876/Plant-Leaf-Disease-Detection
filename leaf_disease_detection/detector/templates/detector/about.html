About LeafScan

Deep-learning plant disease detection using the New Plant Diseases Dataset

Dataset

Built on the New Plant Diseases Dataset from Kaggle. It contains 87,000+ RGB leaf images across {{ total_classes }} classes of 13 crop species — both healthy and diseased samples.

View on Kaggle

Model Architecture

A ResNet50 backbone pre-trained on ImageNet is fine-tuned on the dataset. Input images are resized to 224 × 224 pixels and normalised to [0, 1]. The model outputs a softmax probability distribution over all {{ total_classes }} classes.

Tech Stack

Backend: Django 4.x (Python)
ML / AI: TensorFlow 2.x / Keras (ResNet50)
Frontend: HTML5, CSS3, Vanilla JavaScript
Database: SQLite (zero configuration)
Image Processing: Pillow, NumPy
